Tentang Saya

Saya adalah siswa RPL di SMK Telkom Malang dengan fokus pada backend development. Selama beberapa tahun belajar coding, saya menemukan bahwa bagian paling menarik dari sebuah aplikasi bukan yang terlihat — tapi logika dan sistem yang bekerja di balik layar.

Saya fokus membangun REST API yang clean, merancang struktur database yang efisien, dan memastikan data mengalir dengan aman antara server dan client. Saat ini terbuka untuk kesempatan magang atau kerja sebagai Backend Developer.
